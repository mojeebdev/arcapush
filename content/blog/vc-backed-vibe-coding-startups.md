## Why This List Exists

There is no single authoritative source tracking which vibe coding companies have received venture backing, at what valuations, and from which investors. Vibestream is building that source. This post is the living record.

We define a **VC-backed vibe coding startup** as any company that has received institutional venture investment and whose primary product enables or powers the vibe coding workflow — AI-assisted app building, code generation, or AI-native development environments.

---

## The Complete List

### Cursor (Anysphere)

**Valuation**: $29.3B  
**Category**: AI Code Editor  
**Lead Investors**: Andreessen Horowitz, Accel  
**Also backed by**: Thrive Capital, DST Global  
**What it builds**: An AI-first code editor where developers prompt their way through codebases. The dominant tool for technical vibe coders. Cursor became the fastest-growing developer tool in history by daily active users in 2025.

---

### Vercel

**Valuation**: $9.3B  
**Category**: Deployment & Frontend Infrastructure  
**Lead Investors**: Accel, GV (Google Ventures)  
**Also backed by**: CRV, Geodesic Capital  
**What it builds**: The deployment platform for AI-generated frontends. As vibe coders ship more products, Vercel becomes the default host. The v0 product — which generates full UI from prompts — makes Vercel a direct vibe coding tool.

---

### Lovable

**Valuation**: $6.6B  
**Category**: Full-Stack AI App Builder  
**Lead Investors**: Alphabet (CapitalG), Nvidia, Accel, Khosla Ventures  
**What it builds**: Describe a product in plain English. Lovable generates the full frontend, backend, and database. The most accessible vibe coding tool for non-technical founders. Reached 1M users faster than almost any developer tool in history.

---

### Replit

**Valuation**: $3B  
**Category**: AI-Powered Collaborative IDE  
**Lead Investors**: Andreessen Horowitz, Google  
**Also backed by**: Khosla Ventures, Y Combinator  
**What it builds**: A browser-based coding environment with AI assistance built in. Strong with education, beginners, and rapid prototyping. The Replit Agent feature allows full app generation from prompts.

---

### Bolt (StackBlitz)

**Category**: In-Browser Full-Stack Builder  
**Lead Investors**: Details not yet disclosed  
**What it builds**: Instant full-stack apps in the browser. No setup, no configuration. Bolt became viral in late 2024 for the speed at which non-technical users could ship working applications.

---

### Windsurf (Codeium)

**Category**: AI Code Editor  
**Investors**: General Catalyst, Kleiner Perkins  
**What it builds**: An AI-native code editor competing directly with Cursor. Known for its "flow" state — an agentic coding mode that takes longer tasks end-to-end autonomously.

## What the Data Shows

Looking across these investments, three patterns emerge:

**Infrastructure wins first.** The highest valuations — Cursor at $29.3B, Vercel at $9.3B — go to companies building the picks-and-shovels infrastructure for vibe coders, not just the tools themselves.

**Speed of adoption is unprecedented.** Lovable reached 1M users faster than most developer tools in history. The vibe coding audience is massive and underserved.

**Every major VC firm has a position.** Andreessen Horowitz, Accel, Google Ventures, Khosla Ventures, Nvidia — the top-tier firms are not watching from the sidelines. They are heavily invested.
